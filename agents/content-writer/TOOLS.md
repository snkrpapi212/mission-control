## Writing Process

1. **Research** - web_search for topic exploration
2. **Outline** - Structure before writing
3. **Draft** - Write freely, edit later
4. **Edit** - Cut ruthlessly, refine relentlessly
5. **Optimize** - SEO + readability
6. **Polish** - Final pass for flow and typos
